# 🚀 Chapter 2-2. 데이터 타입

## 🔥 스칼라 타입
  
> 스칼라 타입은 하나의 값을 표현하는 타입을 의미한다.  
  
rust는 정수, 부동소수, 불리언, 문자의 4가지 종류의 타입을 정의하고 있다.  
  
🧨 정수 타입  
|size|signed|unsigned|
|:---:|:---:|:---:|
|8bit|i8|u8|
|16bit|i16|u16|
|32bit|i32|u32|
|64bit|i64|u64|
|arch|isize|usize|  
  
isize와 usize는 현재 아키텍처가 32bit면 32bit, 아키텍처가 64bit면 64bit가 되는 타입이다.  
정수형의 기본 타입은 무조건 i32이다.  
  
🧨 부동소수 타입  
  
32bit는 f32, 64bit는 f64로 표현한다.
부동소수형의 기본 타입은 무조건 f64이다.  
  
🧨 불리언 타입  
  
bool로 표현한다. true와 false의 값중 하나만 표현할 수 있다.  
  
🧨 문자 타입  
  
char로 표현한다. 문자 리터럴은 작은따옴표(') 로 표현한다.  
  
<u>rust의 char는 다른 언어들과 달리 4byte 이다.</u>  
그 이유는, ASCII문자만 표현하는 것이 아닌, 유니코드 문자들도 표현할 수 있기 때문이다.  
하지만, 유니코드의 모든 문자를 표현하지는 못한다.  
  
```rust
fn main() {
    let c = 'z';
    let c = 'Z';
    let balloon = '🎈';     // 컴파일 성공!
}
```

## 🔥 컴파운드 타입  
  
> 컴파운드 타입은 하나의 타입으로 여러 개의 값을 그룹화한 타입이다.  
  
rust는 기본적으로 튜플과 배열 두 가지 컴파운드 타입을 정의하고 있다.  
  
🧨 튜플 타입  
  
튜플은 <u>고정된 길이를 가지며, 한 번 정의하면 그 크기를 키우거나 줄일 수 없다.</u>  
생성할 때는 괄호() 안에 값의 목록을 쉼표로 구분해서 표기한다.  
  
타입 애노테이션을 사용해서 괄호안에 어떤 타입들이 들어갈지 정할 수도 있다.  
  
값을 초기화할 때 괄호 안에 쉼표로 튜플 안의 값들을 각각 받을수도 있다.  
  
```rust
fn main() {
    let tup: (i32, f64, u8) = (500, 6.4, 1);  

    let (x, y, z) = tup; // x, y, z에 각각의 요소들이 들어간다.

    let first = tup.0; // 인덱스로 요소를 가져오는 방법
    let second = tup.1;
    let third = tup.2;
}
```
  
🧨 배열 타입  
  
배열도 <u>고정된 길이를 가지며, 한 번 정의하면 그 크기를 키우거나 줄일 수 없다.</u>  
튜플과 다른 점은 요소들의 타입이 모두 다 같은 타입들이어야 한다.  
  
생성할 때는 대괄호 안에 값의 목록을 쉼표로 구분해서 표기한다.  
  
타입 애노테이션을 사용해서 대괄호안에 어떤 타입이 들어갈지, 요소의 개수가 몇개인지 정할 수 있다.
  
배열은 데이터를 스택에다가 할당하려거나, 항상 고정된 개수의 요소들을 다룰 때 유용하다.  
  
```rust
fn main() {
    let test = [1, 2, 3];
    let test2: [i32; 5] = [1, 2, 3, 4, 5]; // i32타입의 요소가 5개가 있다고 배열 타입 정의
      
    let first = a[0]; // 인덱스로 요소를 가져오는 방법
    let second = a[1];

    let index = 1;
    let idx = a[index];  
      
    let test3 = [1, "아", 2]; // 컴파일 에러!  
}
```
