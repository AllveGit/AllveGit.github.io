# 🚀 Chapter 5-2 : 배열

> 배열은 같은 타입의 객체를 메모리에 일렬로 차례대로 모아놓은 다음,  
메모리 블록을 만들고 그 블록에 이름을 붙인 것이다.
  
## 🔥 배열 생성 및 초기화

> 배열 초기화는 배열이 메모리 블록을 만든 다음에 각 요소에 값을 설정하는 것을 의미한다.  
  
🔔  
배열 초기화는 메모리 블록에 이름을 붙이기 전에 수행된다.  
배열 이름은 r-value로만 사용되는 이유이기도 하다.
{: .notice}  
  
배열 초기화를 하는 방식은 이렇다.  
TYPE ArrayName[N] = { IV0, IV1, ..., IVK }; // (K < N)  
  
N은 배열 요소의 개수이고, IV는 각각 요소의 인덱스와 대응되는 값이다.  
  
```c++
void main()
{
    int arr1[10] = {1};
    int arr2[10] = {1,};

    int arr3[10] = {0};
    int arr4[10] = {};

    int arr5[10] = {1, 2, 3};
    int arr6[10] = {1, 2, 3,};

    int arr7[] = {1, 2, 3, 4};
}
```
  
배열을 초기화하는 여러가지 방식을 사용한 예제코드이다.  
  
arr1과 arr2는  
첫번째 인덱스 0의 요소만 1로 초기화되고, 나머지는 0으로 초기화된다.  
  
arr3과 arr4는  
모든 요소가 0으로 초기화된다.  
    
arr5와 arr6는  
3개의 요소가 차례대로 1, 2, 3으로 채워지고 나머지는 0으로 초기화된다.  
  
arr7은  
지금까지 소개한 배열 초기화 하는 포맷이랑 다르다.  
요소를 처음에 설정하지 않고 초기화값만 넣는다.  
이 초기화 방식은 <u>초기화 값의 개수가 실제 배열 요소의 개수로 설정</u>된다.  
  
<u>모든 요소를 특정한 값으로 채우고 싶으면 그 요소 개수만큼 초기화를 해주어야 한다. </u>  
  
🔔  
배열 원천타입이 1바이트의 크기를 가진다면 memset을 이용하여 한번에 특정한 값을 설정해줄수도 있다.
{: .notice}
