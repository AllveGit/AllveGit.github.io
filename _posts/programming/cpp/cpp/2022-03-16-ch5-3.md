# 🚀 Chapter 5-3 : 다차원 배열

> 배열의 요소 타입은 모든 타입이 올 수 있다. 즉, 배열의 요소 타입으로 배열 타입이 오는 것이 가능하다는 의미이다.  
  
즉, 배열 안에 배열을 집어넣음으로써, 다차원 배열을 구현할 수 있다.

## 🔥 2차원 배열 구조

```c++
void main()
{
  int arr[2][3] = {{11, 12, 13}, {21, 22, 23}}
}
```

위 예제코드는 2차원 배열을 생성 및 초기화한 코드이다.  
의미는 int 객체 3개가 모인 배열 객체를 2개 모아놓은 것이다.    

❗  
int 객체 2개가 모인 배열 객체를 3개 모아놓았다고 해석하면 안된다!  
순서는 항상 올바르게 인지하고 있어야 한다.  
뒷 배열요소 개수가 모인 배열 객체를 앞 배열요소 개수만큼 모아놓았다고 해석해야 한다.
{: .notice--warning}  
  
이런 다차원 배열을 행렬(Matrix) 와 같은 구조로 생각할수도 있다.  
하지만 프로그래밍에서 다루는 메모리, 즉 가상 메모리는 일렬로 쭉 나열된 선형적인 구조를 지니고 있다.  
  
그렇기에, 메모리 적재 구조상 행렬같은 구조로 다차원 배열을 표현할 수 없다.  
<u>다차원 배열은 몇차원이던간에 무조건 선형으로 요소가 이어진다.</u>

배열 이름은 무조건 r-value로만 쓰일 수 있다고 하였는데,  
예제 코드에서의 <u>arr[0]과 arr[1]도 마찬가지로 배열을 나타내므로 r-value로만 사용될 수 있다.</u>  
## 🔥 포인터 대응
  
> 포인터에서 첨자 연산자([])를 사용하면 배열처럼 사용할 수 있다.  
하지만 포인터와 배열은 서로 다른 구조를 가지고 있다는 걸 명심해야 한다.
